# ECommerce-1
Анализ пользовательской базы и операций FreshBox - сервиса доставки свежих фермерских продуктов на дом

1. Код на Python загружает данные из 4 таблиц (customers, orders, order_items, products)
2. Рассчитывает ключевые метрики:
  - Общая выручка за период
  - Средний чек
  - Количество заказов
  - Количество уникальных клиентов
  - Новые vs постоянные клиенты (по дате регистрации)
3. Анализирует выручку по категориям - топ-5 категорий по выручке
4. Анализирует каналы привлечения - какой канал приносит больше клиентов и денег
5. Находит аномалии - заказы с подозрительными характеристиками
6. Строит визуализации:
  - Динамика выручки по месяцам
  - Распределение выручки по категорий
  - Распределение чеков
7. Экспортирует Excel-отчёт с тремя листами:
  - `summary`- ключевые метрики
  - `categories`- анализ по категориям
  - `anomalies`- список подозрительных заказов
